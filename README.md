# viewer-app

## Getting Start

### Prerequisites

Install [Grunt](http://gruntjs.com/)

```bash
$ cd view-app
$ npm install -g grunt-cli
```

### Build

```bash
$ npm install
$ grunt
```

### Start Server

```bash
$ node serve.js
```

Use browser to open 'http://localhost:3001'.
