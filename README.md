# viewer-app

## Getting Start

### Prerequisites

Install [Grunt](http://gruntjs.com/)

```bash
$ cd viewer-app
$ npm install -g grunt-cli
```

### Build

```bash
$ npm install
$ grunt
```

### Grunt-Watch

Run the command and grunt will automatically lints (static code analyze), minimize javascript and css files.

```bash
$ grunt dev
```

### Start Server

```bash
$ node serve.js
```

Use browser to open 'http://localhost:3001'.
